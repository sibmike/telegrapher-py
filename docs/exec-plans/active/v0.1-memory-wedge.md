## Risks & mitigations

| Risk | Mitigation |
|------|-----------|
| Fine-tuned weights not ready when v0.1 wants to ship | MockRunner gates the entire Phase 1–5 dev loop; release waits on weights, not on code |
| `vllm` Windows compatibility issues | `LlamaCppRunner` covers Windows/macOS/CPU; `vllm` is Linux+CUDA only by extras gating |
| LangChain API drift (we target `langchain-core>=0.2`) | Minimum-version pin + tests against the pinned floor |
| Real weights too large for users to download casually | 4-bit quantized GGUF (~5 GB) is the default download path |
